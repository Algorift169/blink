01 / PACKAGED

Install Blink Terminal Emulator v1.0.0

For Debian, Ubuntu, and compatible Linux distributions. This package installs Blink v1.0.0 directly.

shell 
wget https://github.com/Algorift169/blink/releases/download/v1.0.0/blink-v1.0.tar.gz
 tar -xzf blink-v1.0.tar.gz
 cd blink-v1.0 && chmod +x install.sh
 ./install.sh
02 / SOURCE

Build Blink yourself

Clone the repository and compile with C++20. You will need GTK, VTE, pkg-config, and make installed.

shell 
git clone https://github.com/Algorift169/blink.git
cd blink
./scripts/deps.sh
make
./scripts/run.sh

# Feature Notes

## Current capabilities

- Custom decorated window with rounded corners.
- Terminal shell startup through a pseudo-terminal.
- Scrollback and basic terminal font/theme configuration.
- Left-click text selection with VTE.
- Right-click context menu with copy, paste, select all, zoom in, zoom out, and
	zoom reset.
- Clipboard keyboard shortcuts: `Ctrl+Shift+C` and `Ctrl+Shift+V`.
- PTY forwarding for VTE-generated mouse input used by TUIs such as lazygit.
- Search, clipboard, and utility helpers for future expansion.

## Next ideas

- Add tabbed sessions.
- Improve settings persistence.
- Support richer search and copy/paste workflows.

For interaction details, see [Terminal Interactions](terminal-interactions.md).

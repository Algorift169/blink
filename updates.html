Key features

  • Performance-first rendering — FTXUI-powered terminal UI optimized for smooth, low-latency text display.
  • Full PTY integration — Native shell support with accurate terminal behavior, signal handling, and compatibility for modern CLI workflows.
  • Advanced clipboard support — Reliable copy/paste with Ctrl+Shift+C / Ctrl+Shift+V and smart text selection.
  • Mouse support — Terminal mouse reporting for interactive applications, multiplexers, and text selection.
  • Minimal desktop chrome — Lightweight window decoration with a clean titlebar and modern GTK integration.
  • Compact native footprint — C++ implementation focused on fast startup time and minimal resource usage.
